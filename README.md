# Food-Image-Classification
A deep-learning food classification system using ResNet18 trained on 101 food categories. Includes full training pipeline, evaluation metrics, Grad-CAM visualizations, and an interactive Streamlit app for image predictions.
Project Description :
- Deep-learning model for classifying 101 types of food
- Based on ResNet18 
- Training pipeline, evaluation metrics, and Grad-CAM
- Streamlit app for interactive demo
- Dataset used (Food-101 or custom subset)

Features :
- ResNet18 fine-tuned for 101 food classes
- Complete training pipeline (data loading → augmentation → training → evaluation)
- Best model checkpoint included
-  Confusion matrix + F1-score results
-  Grad-CAM visualizations
-  Streamlit app for real-time predictions

Installation packages:
- torch
- torchvision
- streamlit
- matplotlib
- pillow
- scikit-learn
